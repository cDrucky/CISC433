This project finalized hours of hardwork to show a fully functioning computer inside your computer. It is capable of running any program written in the Hack LanguagE (within reason). This computer has working ALU and RAM (Memory and CPU) and is knitted together in the Computer.hdl. Students will enjoy writting different Hack programs and running them on each other machines. This would be an acceptable project in any of the following classes: Essential Algorithms, Operating Systems, Programming III.
